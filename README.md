# Color Game (Java AWT)

This is a simple Java GUI application built using AWT. The program allows users to click different buttons to change the background color of a text field. It is a basic project that demonstrates event handling and layout management in Java.

## Features

* Interactive graphical user interface using Java AWT
* Buttons for multiple colors: Blue, Green, Yellow, Red, Cyan, Orange, Grey
* A Clear button to reset the text field
* Displays the selected color name in the text field
* Changes the background color dynamically

## Technologies Used

* Java
* AWT (Abstract Window Toolkit)
* Event Handling using ActionListener
* GridBagLayout for arranging components

## How to Run

1. Clone the repository:

2. Compile the program:
   javac ColorGame.java

3. Run the program:
   java ColorGame

## Project Structure

ColorGame.java   - Main source file containing GUI and logic
README.md        - Documentation file

## How It Works

When the program runs, a window appears with a text field and several buttons. Each button represents a color. When you click a button, the text field displays the color name and its background changes to that color. The Clear button resets the text and sets the background back to white.

## Concepts Covered

* AWT components like Frame, Button, and TextField
* Layout management using GridBagLayout
* Event-driven programming in Java
* Handling user actions with ActionListener
